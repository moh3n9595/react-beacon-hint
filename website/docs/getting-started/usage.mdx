Here is a simple usage of the **React Beacon Hint**. For more examples please take a look at the [**demo page**](https://react-beacon-hint.vercel.app/).

```jsx
import {Hint} from 'react-beacon-hint';
import 'react-beacon-hint/styles.css';

const Component = () => {
	return (
		<Hint popover='Yay! I appeared!'>
			<button>Click The Hint</button>
		</Hint>
	);
};
```
