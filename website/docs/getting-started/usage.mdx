Here is a simple usage of the **React Beacon Hint**.

```jsx
import {Hint} from 'react-beacon-hint';

const Component = () => {
  return (
  	<Hint popover='Yay! I appeared!'>
      <button>Click The Hint</button>
    </Hint>
  );
};
```
